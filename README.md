## Introduction

透過純 HTML、CSS、JavaScript 實作視差滾動互動式網站，透過滑鼠滾動帶來特別的視覺體驗。

## Use Technology & Library

- ScrollTrigger
- CSS Animations
- Font Awesome

## Features

- [x] 滑鼠滾動觸發動態效果
- [x] 滑鼠移動位置方向產生視差效果
- [x] 判斷裝置只使用直式方向瀏覽畫面
- [x] RWD 版面樣式
